Command prompt

Basic Commands

  1. cd:
    • Ihis command is used to know in which directory you are in.
  2. dir:
    • This command is used to show all the directories.
  3. cd ..:
    • This command is used to go up one directory
  4. cd dir name:
    • Using this command we jump to that directory we want.
  5. dir/w:
    • This command shows names of all the directory.
  6. mkdir dir:
    • This command is used to make new directories.
  7. rmdir dir:
    • To remove an empty directory, we use this command.
  8. help:
    • The help command shows some of the very basic commands.

Safe and Unsafe Commands

safe commands are like:

(a-z)(A-Z)(-,_,.)

unsafe commands are like:

(Everything else)
