setting up git

to set up Git using Command interface we give this commands to to the terminal to register our name and E-mail

  • %git config --global user.name "Abdul Rahim"
  • % git config --global user.email Abdulrahim.Alizada@powercoders.org

To check if the informations are correct we give this command

  • %git config --global user.name
  • % git config --global user.email

Git fundamentals

Three locations: working directory, index, repository.

Git commands copy files between these locations.
